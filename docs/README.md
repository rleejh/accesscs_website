UBC AccessCS is a community dedicated to creating a welcoming space in UBC's Computer Science department for everyone! This is the club's official website, built as a static Vue.js application. It provides information about the club, events, members, and more.

## Features
- Fully static and lightweight
- Responsive design, optimized for mobile and desktop
- Event display and registration
- Accessible design following inclusive web standards
- Contact form

## Technologies Used
- **Frontend:** Vue.js 3
- **Build Tool:** Vite
- **Styling:** Tailwind CSS
- **Languages:** HTML, CSS, JavaScript, Typescript
